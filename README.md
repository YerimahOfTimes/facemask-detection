# 🧠 Face Mask Detection System (ML + FastAPI + Docker)

An end-to-end **Machine Learning project** that detects whether a person is wearing a face mask or not using a CNN model.  
The system is deployed using **FastAPI** and fully containerized with **Docker** for production-ready deployment.

---

## 🚀 Project Overview

This project performs real-time face mask detection using a trained deep learning model.  
It includes:

- Face detection using OpenCV Haar Cascade
- CNN-based mask classification (PyTorch)
- FastAPI backend for prediction API
- Dockerized deployment for portability

---

## 📊 Dataset

- Total Images: 7,553
- Training: 6,042
- Testing: 1,511
- Classes:
  - `with_mask`
  - `without_mask`

---

## 🧠 Model Architecture

- Convolutional Neural Network (CNN)
- PyTorch framework
- Achieved:
  - **Training Accuracy:** ~99%
  - **Test Accuracy:** ~93%
